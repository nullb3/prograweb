ARCHIVO DE PROYECTOS Y EJERCICIOS PREVIOS

Compilación de proyectos y ejercicios realizados, tanto de las tecnologías FrontEnd y BackEnd.



Clase3

Durante las primeras 3 semanas, desarrollamos una solución web para la creación de publicaciones en un foro, utilizando parámetros regulados por campos mediante Postman con una request o método POST. Además, somos capaces de recuperar todas las publicaciones creadas con una request o método GET.

Por supuesto, contamos con implementación de LINQ en C Sharp como lógica. Además de montar una base de datos preexistente y utilizar la conexión de Postman a un API también personalizada.

Clase4

Se trata de una nueva implementación de Clase3, donde esta vez podemos crear usuarios para un sistema de login, y dichos usuarios pueden logearse al sistema.

Por supuesto, contamos con implementación de LINQ en C Sharp como lógica. Además de montar una base de datos preexistente y utilizar la conexión de Postman a un API también personalizada.

Clase5

Un simple y sencillo pero funcional sitio web de Introducción, nos sirvió como dummy para entender la estructura de HTML y crear además, un formulario utilizando las capacidades de HTML5, que, por increíble que parezca, posee lógica.

Clase7

Una mejora de Clase5, donde utilizamos por primera vez CSS para mejorar la apariencia rígida de nuestro HTML.

Creamos tablas y elementos interactivos decorados sin el uso de lógica, es decir, sin implementar JavaScript aún.

Clase7Bootstrap & Clase8

Una solución construída sobre Clase7 que nos permite implementar el framework de Bootstrap en nuestro sitio web, facilitando la creación de interfaaces interactivas y mejorando considerablemente la profesionalidad de nuestros sitios.

Implementa barras de navegación, carruseles de imágenes y tarjetas interactivas, además de footers elegantes.

ExamenPrograWeb1

Se trata de un sistema de notas para estudiantes, ¡Donde entra en juego todo lo visto para la creación del BackEnd!

Posee procedimientos para el registro de estudiantes, creación de calificaciones por estudiante y la capacidad de visualizar todas las notas ingresadas.
