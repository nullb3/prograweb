BIENVENIDO A LA SECCIÓN DE FRONTEND

Encontrarás los conocimientos adquiridos en HTML, CSS3 y Bootstrap hasta la fecha.



HTML

Esta es la primera tecnología que exploramos en la segunda parte del curso, se trata de HTML o Hyper Text Markup Language. Es, esencialmente, el esqueleto de nuestro sitio web, ya que le da forma y rigidez a los elementos en el mismo.

Ejemplo de HTML

Aquí puedes ver un ejemplo de código HTML utilizado en el aprendizaje de FrontEnd:


            <!DOCTYPE html> <!--Indicates that this is an HTML5 page, without plugins-->
            <html lang="en"> <!--Indicates the page language-->
            <head>
                <title>PrograWeb Semana 5</title>
                <script src="core10.js"></script>
            </head>
            <body> 
                <h1 style="text-align: center;">Welcome to Index</h1>
                <h6><marquee behavior="alternate">This is a &lt;Marquee&gt;</marquee></h6>
                <p style="text-align: center;">This is a centered paragraph.</p>
                <p style="text-align: center;"><a target="_blank" href="https://www.ulatina.ac.cr">Ulatina.ac.cr</a></p>
                <img style="display: block; margin: auto;" height="50" width="50" alt="Hi, this is the .NET logo" src="resources\NET.icon.png">
                <p style="text-align: center;"><a href="form.html">Ir a Formulario</a></p>
            </body>
            </html>

CSS

Sin embargo, la tecnología de HTML es muy simple a la vista, ahí es donde entra CSS o Cascading Style Sheets, un lenguaje de diseño gráfico que nos permite embellecer nuestras páginas web para que luzcan, justamente, como la que estás leyendo ahora mismo.

Ejemplo de CSS

Aquí puedes ver un ejemplo de código CSS utilizado en este mismo sitio web:


            .code-section{
                background-color: #000;
                color: #6ff;
                padding: 20px;
                border-radius: 8px;
            }

            .code-section h2{
                color: #6ff;
            }

            .code-section pre{
                background-color: #111;
                color: #6ff;
                padding: 15px;
                border-radius: 8px;
            }

Bootstrap

Hasta el momento, la tercer tecnología que hemos empleado es Bootstrap, desarrollada por el equipo de Twitter. Se trata de un framework de CSS, es decir, una versión personalizada de CSS que cuenta con todo lo necesario para crear sitios web modernos, agradables a la vista y "Responsive".

Funciona principalmente con contenedores que nos permiten acceder a un sistema de 12 columnas universales que dan uniformidad al sitio web.
